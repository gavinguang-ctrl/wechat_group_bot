# 微信群自动化机器人 (WeChat Group Bot)

基于 `wxauto` 的微信 UI 自动化工具，支持自动扫码进群和定时 AI 改写发消息。

## 核心功能
- **自动进群**：扫描最近日期文件夹下的群二维码，发送到文件传输助手，手机端确认后自动记录。
- **手动加群**：直接输入微信群名称，将其纳入发送计划，无需二维码。
- **AI 智能改写**：对接 OpenAI/Gemini/DeepSeek 等接口，根据群名自动改写文案，避免由于内容雷同被封。
- **定时发送**：支持设置发送间隔（如 1-3 天），自动轮换账号。
- **安全风控**：模拟人工 UI 操作，非协议版，大幅降低封号风险。

## 快速开始
1. **安装环境**：确保已安装 Python 3.8+。
2. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```
3. **配置文件**：编辑 `config.py`，填写你的 AI API Key 和二维码路径。
4. **启动程序**：
   - **双击 `start.bat`** 打开可视化控制面板（推荐）。
   - 或者直接运行 `python main.py`。

## 控制面板菜单
双击 `start.bat` 后可选择：
- **[1] Full Run**: 一键执行全流程（扫码 → 进群 → 发消息）
- **[2] Scan and Join**: 仅扫码进群（扫描最近日期文件夹）
- **[3] Send Messages**: 启动定时发消息循环
- **[4] View Group Status**: 查看群数据库状态
- **[5] Open QR Folder**: 打开二维码文件夹
- **[6] Edit Template**: 编辑消息模板
- **[7] Manually Add Group**: 手动添加群聊到发送列表

## 文件说明
- `start.bat`: 一键启动控制面板（推荐使用）
- `main.py`: 主程序流程
- `config.py`: 所有参数配置（API Key、路径、风控频率）
- `scan_join.py`: 扫码进群（仅扫最近日期文件夹）
- `send_loop.py`: 定时轮换发消息
- `add_group.py`: 手动添加群聊
- `status.py`: 查看群数据库状态
- `group_db.json`: 本地数据库，记录进群状态和发送进度

## 注意事项
- 运行前请确保 PC 版微信已登录且窗口未被完全最小化。
- 扫码进群时，请在手机微信【文件传输助手】中逐张长按识别二维码。
- 本工具仅供学习参考，请在相关法律法规允许的范围内使用。
